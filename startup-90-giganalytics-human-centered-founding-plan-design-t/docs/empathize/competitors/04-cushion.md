# Cushion — Competitor Analysis

**URL:** https://cushionapp.com/  
**Analyzed:** April 2026  
**Category:** Freelance Income Planning / Scheduling

## Overview
- **Target User:** Independent freelancers who struggle with feast-or-famine income cycles and overbooking
- **Pricing:** $16/month or $144/year
- **Core Value Prop:** "Peace of mind for freelancers" — visualize your year ahead, set income goals, avoid overbooking

## Key Features
- Year-at-a-glance availability timeline (visual calendar of booked/available periods)
- Income goal setting (minimum, target, stretch goals)
- Invoice tracking and payment status
- Client insights (who pays on time, who delays, who's worth keeping)
- Project scheduling with drag-and-drop
- Potential income view (sum of pipeline opportunities)
- Safe-to-take-break indicator
- Delay and deadline tracking

## UX Patterns
- **Onboarding:** Add your first project → set income goal → see your year timeline immediately; fast value delivery
- **Data Input:** Manual project entry with dates and rates; no automatic import
- **Visualization:** Horizontal timeline showing months ahead with color-coded work density; unique and highly visual
- **Income Dashboard:** Three goal bars (minimum/target/stretch) with progress; "you're safe to take a break" messaging
- **Client Insights:** Per-client stats on payment speed, delays, project count

## Strengths
- Most visually intuitive scheduling tool for freelancers
- Goal-setting with three tiers is psychologically smart (minimum vs. target vs. stretch)
- Client health metrics (who pays late) is uniquely valuable
- "Safe to take a break" feature is emotionally resonant
- Built specifically and only for independent freelancers

## Weaknesses / Gaps
- **Manual data entry only** — no bank sync, no API import, no automation
- **No time tracking** — no connection between time spent and income earned
- **No multi-platform analytics** — doesn't distinguish Upwork earnings from Fiverr
- **No true hourly rate** — tracks project value, not time invested
- **No marketplace fee modeling**
- **No benchmark data** — no comparison to peers
- **Scheduling only, not analytics** — tells you what's booked, not what's performing
- Small feature set limits retention over time

## UX Flows (Annotated)
1. **Year-view flow:** Dashboard lands on full-year timeline → months color-coded by work density → click month to see projects → drag project block to reschedule. Extremely visual.
2. **Goal-setting flow:** Set minimum income (rent + bills) → target (comfortable) → stretch (ideal) → progress bars update as invoices are added. Satisfying UX.
3. **Client insight flow:** Client list → click client → see avg. days to pay, projects completed, delays, total value. Actionable intelligence.

## Relevance to GigAnalytics
- **Closest competitor in spirit** — Cushion targets the same emotional need (income stability, planning) but lacks automation and analytics depth.
- **Borrow:** Year-view timeline for availability/scheduling, three-tier income goals, client health metrics.
- **Differentiate:** Add automation (bank sync, platform API), time tracking, true hourly rate, multi-platform comparison. Cushion is what GigAnalytics should make obsolete.
- **Key insight:** Cushion's testimonials reveal the core pain — "helps me understand where income is coming from and when it's coming in." That's GigAnalytics' mission statement.
