**Text (comment body after submitting):**
```
I kept seeing developers hand-roll Stripe + SSG setups to sell technical courses. TeachRepo automates this: write lessons in Markdown, add a course.yml with pricing and structure, push to GitHub, and get a deployed course with Stripe checkout and auto-graded quizzes.

Free tier: deploy to your own Vercel/server, keep 100% of revenue.
Hosted tier: $19/mo, 5% platform fee, marketplace listing.

Key features:
- Markdown-native authoring (lessons live in the same repo as your code)
- Stripe checkout, zero config
- Auto-graded quizzes via YAML frontmatter  
- Gated StackBlitz/CodeSandbox embeds unlocked on purchase
- Git-versioned — every push updates your course with full version history
- Built-in affiliate/ref tracking

Tech: Next.js 15 App Router, Supabase, Stripe, MDX, Playwright E2E suite (600+ tests).

Two free sample courses are live on the marketplace: "Git for Engineers" and "GitHub Actions for Engineers".

Happy to answer questions about the architecture, the self-hosting path, or the quiz grading system.
```
