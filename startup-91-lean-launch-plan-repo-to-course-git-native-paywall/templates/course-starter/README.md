```
lessons/
├── 01-introduction.md

└── 03-advanced-topics.md
```

## Publishing

```bash
# Install TeachRepo CLI
npm install -g @teachrepo/cli

# Preview locally
teachrepo dev

# Publish
teachrepo publish
```
