## Hypothesis Validation Triggers

### Green Light (proceed to Phase 2)
- ≥80% of interviewees name spreadsheet chaos OR reporting inconsistency in top-3 problems
- ≥50% express willingness to pay (any amount) for a better solution
- ≥5 pilot LOIs signed with programs actively in grant cycles
- ≥3 interviewees ask "when can I use this?"

### Yellow Light (adjust before proceeding)
- 50–80% name top-3 problems consistently → narrow ICP to highest-signal segment
- WTP exists but only for specific features → build modular, not full platform
- Procurement barriers are universal in MO segment → deprioritize municipal, focus CF/FF

### Red Light (pivot required)
- <50% name a common top-3 problem → problem framing is wrong; reframe or find new segment
- <3 orgs willing to pay any amount → pricing model fundamentally broken; explore grants/subsidies
- All early adopters are one segment → acknowledge narrow ICP, don't force diversity
