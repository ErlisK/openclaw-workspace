## 3. Data and Security

### 3.1 Data handling
ClaimCheck Studio processes manuscript and content text only. No PHI is required or accepted.
Open-access article snapshots are stored in private encrypted storage (Supabase, AES-256, us-east-1).
Subscriber/paywalled content is processed in-memory only — no full text stored.

### 3.2 Data retention
Session data retained for 90 days post-session by default. Extended retention available on request.
Audit trail retained for compliance purposes (immutable, exportable).

### 3.3 Confidentiality
Both parties agree to keep the terms of this MoU confidential.
ClaimCheck Studio will not publish Pilot Partner's name or results without written consent.
Pilot Partner may reference ClaimCheck Studio in internal communications.

### 3.4 Sub-processors
Vercel (hosting, SOC 2 Type II), Supabase (database + storage, SOC 2 Type II), 
Stripe (billing, PCI DSS Level 1), AWS Bedrock (LLM inference, SOC 2, HIPAA eligible).
Full sub-processor list: citebundle.com/security
